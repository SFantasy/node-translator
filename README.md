Yet another Chinese-English translate tool written in Node.

## Install

```
[sudo] npm install ntrans [-g]
```

## Usage

```
$ ntrans chrome

 chrome

 美音: [krom]   英音: [krəʊm]

 翻译：n. 铬，铬合金；铬黄；谷歌浏览器
```
